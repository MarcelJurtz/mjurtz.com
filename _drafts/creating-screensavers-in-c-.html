Today, we will cover how you can create custom screensavers for windows computers using C#.

The goal of this article is to create a simple screensaver which shows the current time and a customizable image or slideshow. You can find the required source code for my implementation on my GitHub profile.

Components of a Screensaver

Before creating a custom screensaver, you need to know the different components a screensaver has in windows. There are basically three different elements that need to be implemented:

  • The screensaver itself
  • The configuration page
  • The preview

You can think of each component as separate entry point of the program. This being said, you can launch each part separately by using arguments:

  • /s - screensaver
  • /c - configuration
  • /p - preview

In my case, I use a standard WinForms application, which covers the argument handling as shown below:

string param = args[0].ToLower().Trim();

if (firstArgument == "/c")
{
    // Settings
}
else if (firstArgument == "/p")
{
    // Preview
}
else if (firstArgument == "/s")
{
    // Screensaver
}
else
{
    MessageBox.Show("Invalid Argument: \"" + firstArgument +"\".", "ScreenSaver", MessageBoxButtons.OK, MessageBoxIcon.Information);
}
